Quick Start

  1. Load a model checkpoint from the dropdown
  2. Choose a generation mode (Unconditional, Conditional, or Inpainting)
  3. Adjust parameters as desired
  4. Click "Generate" and wait for results
  5. Download MIDI, image, or GIF

API Endpoints

GET /api/status

Get system status and model information.

GET /api/models/list

List available model checkpoints.

POST /api/models/load

Load a specific checkpoint.

POST /api/generate

Start a generation job (returns job ID).

GET /api/generate/{job_id}

Check status of a generation job.

WebSocket /ws/generate

Real-time generation with progress updates.

Running the Server

cd web/backend
python app.py

Server will start on http://localhost:8000

Docker Deployment

docker-compose up -d

See docker-compose.yml for configuration.
