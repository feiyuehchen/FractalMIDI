About the Project

What is FractalMIDI?

FractalMIDI is a hierarchical music generation system based on fractal generative models. It generates piano roll music through a multi-level autoregressive process, creating complex musical structures from simple patterns.

Technical Architecture

The model uses a 4-level hierarchy, where each level generates patches that are refined by the next level. This fractal approach allows for both global structure and local detail in the generated music.

Generation Modes

Unconditional Generation

Generate music from scratch with no constraints.

Conditional Generation

Continue from a given musical phrase, maintaining style and structure.

Inpainting

Fill in missing or erased regions while preserving surrounding context.

Model Types

MAR (Masked Autoregressive)

Generates patches through iterative refinement, predicting masked tokens. Generally produces higher quality results.

AR (Autoregressive)

Generates patches sequentially in raster scan order (row-major or column-major). Faster but may be slightly lower quality.
